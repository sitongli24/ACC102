## 1. Project Purpose & Target User
This project analyzes the risk and return performance of Apple, Microsoft, and Amazon stocks using historical price data. It is designed for investors and accounting/finance students to understand return, volatility, and Sharpe ratio.

## 2. Data Source
- Database: WRDS
- Data type: Daily stock prices (2020–2025)
- Access date: [2026.04.20]

## 3. Python Methods Used
- Data loading and cleaning
- Daily return calculation
- Annualized return & volatility
- Sharpe ratio computation
- Data visualization 
- Pandas for data management

## 4. Key Results
- Compared risk and return across three tech stocks
- Visualized price trends and return volatility
- Generated risk-adjusted performance metrics

## 5. Files in Repository
- analysis.ipynb: Full Python analysis notebook
- price_trend.png: Stock price trend chart
- return_volatility.png: Daily return volatility
- stock_analysis_final.png: Risk & return comparison
